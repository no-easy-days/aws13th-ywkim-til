### 변수와 자료형 (int, float, str, bool, list, dict, tuple, set)

- int : 정수형 변수이다.
- float : 부동소수점수를 의미하는 자료형이다.
- str : 'string'의 줄임말로 문자열을 의미한다.
- bool : 'boolean'의 줄임말로 문자열을 의미한다.
- list : 여러개의 데이터를 저장하는 공간으로 '리스트 이름 = [항목1,항목2,...]' 으로 리스트이름의 리스트를 생성한다.
```python
a = []
b = [1, 2, 3]
c = ['Life', 'is', 'too', 'short']
d = [1, 2, 'Life', 'is']
e = [1, 2, ['Life', 'is']]
```
- dict : 딕셔너리 자료형이다. 딕셔너리는 리스트와 마찬가지로 하나 이상의 데이터를 저장하는 자료형으로 '키'와 '값' 쌍으로 이루어졌다. 리스트의 항목에 접근할때는 인덱스를 사용하지만
         딕셔너리의 값에 접근할 때는 키를 사용한다
         딕셔너리 이름 = {키1:값1,키2:값2,...} 로 생성한다.
```python
dic = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
```
- tuple : 리스트와 비슷하다 리스트는 요솟값의 생성, 삭제, 수정이 가능하지만, 튜플은 요솟값을 바꿀 수 없다.
          그리고 ()를 사용해서 튜플을 생성한다
```python
t1 = ()
t2 = (1,)
t3 = (1, 2, 3)
t4 = 1, 2, 3
t5 = ('a', 'b', ('ab', 'cd'))
```
- set : 집합 자료형이다. 중복을 허용하지 않고 순서가 없는 데이터들의 모임으로, 교집합, 합집합, 차집합 등의 집합 연산을 쉽게 처리할 수 있다.
```python
s1 = set([1, 2, 3, 4, 5, 6])
s2 = set([4, 5, 6, 7, 8, 9])
# 교집합
s1 & s2
>>> {4, 5, 6}
# 합집합
s1 | s2
>>> {1, 2, 3, 4, 5, 6, 7, 8, 9}
# 차집합
s1 - s2
>>> {1, 2, 3}
s2 - s1
>>> {8, 9, 7}
```

### 조건문과 반복문 (if, for, while)

- if문은 조건이 만족하면 true 불만족하면 false가 실행되게하는 조건문이다.
```python
fee = 5000
age = int(input('나이 : '))
if age < 8:
  print('입장료 : ' , fee * 0.7)
elif age < 60:
  print('입장료 : ' , fee)
else:
  print('입장료 : ' , fee * 0.8)
```
- for문은 반복문이다
```python
test_list = ['one', 'two', 'three'] 
 for i in test_list: 
     print(i)
 
>>> one 
>>> two 
>>> three
```
- while문도 for문과 같이 반복문이지만 조금의 차이는 조건이 참인동안 while문안에 내용들이 실행된다.
```python
treeHit = 0
while treeHit < 10:
     treeHit = treeHit +1
     print("나무를 %d번 찍었습니다." % treeHit)
     if treeHit == 10:
         print("나무 넘어갑니다.")

나무를 5번 찍었습니다.
나무를 6번 찍었습니다.
나무를 7번 찍었습니다.
나무를 8번 찍었습니다.
나무를 9번 찍었습니다.

나무 넘어갑니다.
```

### 클래스에 대하여
- 클래스 : 파이썬에서 객체는 클래스를 이용해서 생성하는데 쉽게 붕어빵 틀(template)라고 생각하면된다.
- 생성자 : 객체가 생성될때 자동으로 호출되는 메소드로 초기화를 담당한다.
```python
class Account:
  balance =0

  def deposit(self,amount):
    self.balance += amount
    return self.balance

  def withdraw(self,amount):
    self.balance-=amount
    return self.balance

a1 = Account()  #객체 생성하기
print(a1.deposit(100))
>>> 100
print(a1.withdraw(30))
>>> 70

```
